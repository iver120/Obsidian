# Hurtig innføring i algebra

## Hva er algebra?
Algebra er matematikk med bokstaver og tall sammen. Bokstavene (som a, x, y) representerer ukjente tall eller variable verdier.

## Grunnleggende begreper

### Variabler og konstanter
- **Variabel**: En bokstav som representerer et ukjent tall (f.eks. x, y, a)
- **Konstant**: Et fast tall (f.eks. 5, -3, 0)
- **Koeffisient**: Tallet foran en variabel (i 3x er 3 koeffisienten)

### Algebraiske uttrykk
- **Ledd**: Deler av et uttrykk som er adskilt med + eller - (f.eks. i 3x + 5 er 3x og 5 to ledd)
- **Like ledd**: Ledd med samme variabel og potens (f.eks. 2x og 5x)

## Grunnregler for bokstavregning

### Addisjon og subtraksjon
- Bare like ledd kan legges sammen: $2x + 3x = 5x$
- Ulike ledd kan ikke slås sammen: $2x + 3y$ forblir $2x + 3y$

### Multiplikasjon
- Tall × variabel: $3 \times x = 3x$
- Variabel × variabel: $x \times x = x^2$
- Koeffisienter multipliseres: $2x \times 3y = 6xy$

### Divisjon
- $\frac{6x}{2} = 3x$
- $\frac{x^3}{x} = x^2$

## Parentesregler

### Distribusjonsregelen
Når du multipliserer med noe utenfor parentesen:
$$a(b + c) = ab + ac$$

**Eksempler:**
- $3(x + 2) = 3x + 6$
- $-2(3x - 4) = -6x + 8$
- $x(x + 5) = x^2 + 5x$

### Parenteser med fortegn
- $+(a + b) = a + b$
- $-(a + b) = -a - b$
- $-(a - b) = -a + b$

### Flere parenteser
$$(a + b)(c + d) = ac + ad + bc + bd$$

**Eksempel:**
$$(x + 2)(x + 3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$$

## Viktige tips

### Rekkefølge av operasjoner (PEMDAS/BODMAS)
1. **P**arenteser først
2. **E**ksponenter (potenser)
3. **M**ultiplikasjon og **D**ivisjon (fra venstre til høyre)
4. **A**ddisjon og **S**ubtraksjon (fra venstre til høyre)

### Vanlige feil å unngå
- ❌ $2x + 3y = 5xy$ (kan ikke legge sammen ulike ledd)
- ✅ $2x + 3x = 5x$ (bare like ledd)
- ❌ $(x + 2)^2 = x^2 + 4$
- ✅ $(x + 2)^2 = x^2 + 4x + 4$

## Øvingsoppgaver

### Nivå 1: Grunnleggende
1. Forenkle: $3x + 2x - x$
2. Utvid: $4(x + 3)$
3. Forenkle: $2x + 5 - x + 1$

### Nivå 2: Mellom
1. Utvid: $(x + 2)(x + 4)$
2. Forenkle: $3(2x - 1) - 2(x + 3)$
3. Utvid: $-2(3x - 4y + 1)$

### Nivå 3: Utfordrende
1. Forenkle: $2x(x + 3) - x(x - 1)$
2. Utvid: $(2x - 3)(x + 5)$
3. Forenkle: $4(x - 2) - 3(2x + 1) + 5x$
