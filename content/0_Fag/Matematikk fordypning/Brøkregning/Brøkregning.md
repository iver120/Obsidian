## Hva er brøkregning?
Brøkregning handler om å regne med tall som er skrevet som deler av hele tall. En brøk består av en teller (øverst) og en nevner (nederst), der nevneren forteller hvor mange deler helheten er delt inn i.

## Grunnleggende begreper

### Brøkdeler
- **Teller**: Tallet over brøkstreken som viser hvor mange deler vi har
- **Nevner**: Tallet under brøkstreken som viser hvor mange deler helheten er delt i
- **Brøkstrek**: Linjen som skiller teller og nevner
- **Ekte brøk**: Teller mindre enn nevner (f.eks. $\frac{3}{4}$)
- **Uekte brøk**: Teller større enn eller lik nevner (f.eks. $\frac{7}{4}$)

### Felles nevner
For å legge sammen eller trekke fra brøker må de ha samme nevner.

## Regler for brøkregning

### Forkorting og utviding
- **Forkorting**: Dele teller og nevner med samme tall: $\frac{6}{8} = \frac{3}{4}$
- **Utviding**: Gange teller og nevner med samme tall: $\frac{2}{3} = \frac{6}{9}$

### Addisjon med felles nevner
Når nevnerne er like, legger vi sammen tellerne:
$$\frac{2}{7} + \frac{3}{7} = \frac{2+3}{7} = \frac{5}{7}$$

### Subtraksjon med felles nevner
Når nevnerne er like, trekker vi fra tellerne:
$$\frac{5}{8} - \frac{2}{8} = \frac{5-2}{8} = \frac{3}{8}$$

### Finne felles nevner
**Metode 1 - Minste felles multiplum (MFM):**
For $\frac{1}{4} + \frac{1}{6}$: MFM av 4 og 6 er 12
$$\frac{1}{4} + \frac{1}{6} = \frac{3}{12} + \frac{2}{12} = \frac{3+2}{12} = \frac{5}{12}$$

**Metode 2 - Kryss-multiplisering:**
$$\frac{2}{3} + \frac{1}{4} = \frac{2 \cdot 4}{3 \cdot 4} + \frac{1 \cdot 3}{4 \cdot 3} = \frac{8}{12} + \frac{3}{12} = \frac{8+3}{12} = \frac{11}{12}$$

### Addisjon med ulike nevnere
**Eksempel 1:**
$$\frac{3}{5} + \frac{2}{10} = \frac{6}{10} + \frac{2}{10} = \frac{(6)+(2)}{10} = \frac{8}{10} = \frac{4}{5}$$

**Eksempel 2:**
$$\frac{1}{3} + \frac{2}{9} = \frac{3}{9} + \frac{2}{9} = \frac{(3)+(2)}{9} = \frac{5}{9}$$

### Subtraksjon med ulike nevnere
**Eksempel 1:**
$$\frac{7}{8} - \frac{1}{4} = \frac{7}{8} - \frac{2}{8} = \frac{(7)-(2)}{8} = \frac{5}{8}$$

**Eksempel 2:**
$$\frac{5}{6} - \frac{1}{3} = \frac{5}{6} - \frac{2}{6} = \frac{(5)-(2)}{6} = \frac{3}{6} = \frac{1}{2}$$

## Viktige tips

### Alltid forkort til laveste form
Etter regning, sjekk om brøken kan forkortes.

### Vanlige feil å unngå
- ❌ $\frac{2}{3} + \frac{1}{4} = \frac{3}{7}$ (kan ikke legge sammen teller og nevner direkte)
- ✅ $\frac{2}{3} + \frac{1}{4} = \frac{8}{12} + \frac{3}{12} = \frac{11}{12}$
- ❌ $\frac{6}{8} - \frac{2}{4} = \frac{4}{4} = 1$ (må ha felles nevner først)
- ✅ $\frac{6}{8} - \frac{2}{4} = \frac{6}{8} - \frac{4}{8} = \frac{2}{8} = \frac{1}{4}$

## Fasit

### Nivå 1:
1. $\frac{2}{9} + \frac{4}{9} = \frac{6}{9} = \frac{2}{3}$
2. $\frac{7}{10} - \frac{3}{10} = \frac{4}{10} = \frac{2}{5}$
3. $\frac{1}{8} + \frac{3}{8} + \frac{2}{8} = \frac{6}{8} = \frac{3}{4}$
4. $\frac{11}{12} - \frac{5}{12} = \frac{6}{12} = \frac{1}{2}$

### Nivå 2:
1. $\frac{1}{2} + \frac{1}{4} = \frac{2}{4} + \frac{1}{4} = \frac{(2)+(1)}{4} = \frac{3}{4}$
2. $\frac{3}{4} - \frac{1}{8} = \frac{6}{8} - \frac{1}{8} = \frac{(6)-(1)}{8} = \frac{5}{8}$
3. $\frac{2}{3} + \frac{1}{6} = \frac{4}{6} + \frac{1}{6} = \frac{(4)+(1)}{6} = \frac{5}{6}$
4. $\frac{5}{6} - \frac{1}{3} = \frac{5}{6} - \frac{2}{6} = \frac{(5)-(2)}{6} = \frac{3}{6} = \frac{1}{2}$
5. $\frac{1}{5} + \frac{3}{10} = \frac{2}{10} + \frac{3}{10} = \frac{(2)+(3)}{10} = \frac{5}{10} = \frac{1}{2}$

### Nivå 3:
1. $\frac{2}{3} + \frac{3}{4} = \frac{8}{12} + \frac{9}{12} = \frac{(8)+(9)}{12} = \frac{17}{12}$
2. $\frac{5}{8} - \frac{1}{6} = \frac{15}{24} - \frac{4}{24} = \frac{(15)-(4)}{24} = \frac{11}{24}$
3. $\frac{3}{7} + \frac{2}{5} = \frac{15}{35} + \frac{14}{35} = \frac{(15)+(14)}{35} = \frac{29}{35}$
4. $\frac{7}{9} - \frac{1}{4} = \frac{28}{36} - \frac{9}{36} = \frac{(28)-(9)}{36} = \frac{19}{36}$
5. $\frac{1}{4} + \frac{2}{3} - \frac{1}{6} = \frac{3}{12} + \frac{8}{12} - \frac{2}{12} = \frac{(3)+(8)-(2)}{12} = \frac{9}{12} = \frac{3}{4}$

### Nivå 4:
1. $\frac{3}{8} + \frac{5}{12} - \frac{1}{6} = \frac{9}{24} + \frac{10}{24} - \frac{4}{24} = \frac{(9)+(10)-(4)}{24} = \frac{15}{24} = \frac{5}{8}$
2. $\frac{2}{5} - \frac{3}{8} + \frac{1}{10} = \frac{16}{40} - \frac{15}{40} + \frac{4}{40} = \frac{(16)-(15)+(4)}{40} = \frac{5}{40} = \frac{1}{8}$
3. $\frac{7}{15} + \frac{2}{9} - \frac{1}{5} = \frac{21}{45} + \frac{10}{45} - \frac{9}{45} = \frac{(21)+(10)-(9)}{45} = \frac{22}{45}$
