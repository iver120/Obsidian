# Bokstavregning

Bokstavregning er regning med variabler (bokstaver). I stedet for konkrete tall, arbeider vi med generelle størrelser representert ved bokstaver som x, y, a, b osv.

## Grunnleggende regler

### Like termer
Like termer har samme variabel med samme eksponent og kan slås sammen.

**Like termer:**
- $3x$ og $7x$ → $3x + 7x = 10x$
- $4a^2$ og $2a^2$ → $4a^2 + 2a^2 = 6a^2$
- $5xy$ og $3xy$ → $5xy + 3xy = 8xy$

**Ikke like termer:**
- $3x$ og $4y$ → kan ikke slås sammen
- $2x$ og $5x^2$ → kan ikke slås sammen
- $3xy$ og $4xz$ → kan ikke slås sammen

## Addisjon og subtraksjon

**Addisjon:**
- $2x + 5x = 7x$
- $3a + 4b + 2a = 5a + 4b$
- $x^2 + 3x^2 = 4x^2$

**Subtraksjon:**
- $8x - 3x = 5x$
- $7y - 2y = 5y$
- $4a^2 - a^2 = 3a^2$

**Blandet addisjon og subtraksjon:**
- $5x + 3y - 2x + 4y = 3x + 7y$
- $2a^2 + 5a - a^2 + 3a = a^2 + 8a$

## Multiplikasjon

### Konstant ganger variabel
- $3 \times x = 3x$
- $5 \times 2y = 10y$
- $4 \times 3a^2 = 12a^2$

### Variabel ganger variabel
- $x \times x = x^2$
- $y \times y \times y = y^3$
- $a \times b = ab$
- $2x \times 3y = 6xy$

### Multiplikasjon med samme base
- $x^2 \times x^3 = x^{2+3} = x^5$
- $a^4 \times a = a^{4+1} = a^5$
- $3x^2 \times 2x^3 = 6x^{2+3} = 6x^5$

**Viktig regel:** $x^m \times x^n = x^{m+n}$

### Multiplikasjon av ulike variabler
- $x \times y = xy$
- $2a \times 3b = 6ab$
- $x^2 \times y^3 = x^2y^3$

## Potenser

### Grunnleggende potenser
- $x^1 = x$
- $x^2 = x \times x$
- $x^3 = x \times x \times x$
- $x^0 = 1$ (for $x \neq 0$)

### Potens av potens
- $(x^2)^3 = x^{2 \times 3} = x^6$
- $(a^4)^2 = a^{4 \times 2} = a^8$

**Viktig regel:** $(x^m)^n = x^{mn}$

### Potens av produkt
- $(xy)^2 = x^2y^2$
- $(3a)^2 = 3^2 \times a^2 = 9a^2$
- $(2xy)^3 = 2^3 \times x^3 \times y^3 = 8x^3y^3$

**Viktig regel:** $(xy)^n = x^n y^n$

## Divisjon

### Konstant dividert med variabel
- $\frac{6x}{2} = 3x$
- $\frac{12a^2}{4} = 3a^2$
- $\frac{15xy}{5} = 3xy$

### Variabel dividert med variabel (samme base)
- $\frac{x^5}{x^2} = x^{5-2} = x^3$
- $\frac{a^7}{a^3} = a^{7-3} = a^4$
- $\frac{6x^4}{2x} = 3x^{4-1} = 3x^3$

**Viktig regel:** $\frac{x^m}{x^n} = x^{m-n}$

### Spesialtilfeller
- $\frac{x}{x} = 1$ (for $x \neq 0$)
- $\frac{x^3}{x^3} = 1$ (for $x \neq 0$)
- $\frac{1}{x} = x^{-1}$ (negative eksponenter)

## Distributiv egenskap med variabler

### Gange inn i parenteser
- $3(x + 4) = 3x + 12$
- $a(b + c) = ab + ac$
- $2x(3y + 5) = 6xy + 10x$
- $x(x + 3) = x^2 + 3x$

### Gange ut parenteser
- $(x + 2)(x + 3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$
- $(a + b)(c + d) = ac + ad + bc + bd$
- $(2x + 1)(x + 4) = 2x^2 + 8x + x + 4 = 2x^2 + 9x + 4$

## Negative eksponenter (grunnleggende)

- $x^{-1} = \frac{1}{x}$
- $x^{-2} = \frac{1}{x^2}$
- $\frac{1}{x^{-2}} = x^2$

## Vanlige feil å unngå

**Feil:** $x^2 + x^3 = x^5$
**Riktig:** $x^2$ og $x^3$ er ikke like termer

**Feil:** $(x^2)^3 = x^5$
**Riktig:** $(x^2)^3 = x^6$

**Feil:** $x^2 \times x^3 = x^6$
**Riktig:** $x^2 \times x^3 = x^5$

**Feil:** $(xy)^2 = xy^2$
**Riktig:** $(xy)^2 = x^2y^2$

## Sammenfatning av viktige regler

| Operasjon | Regel | Eksempel |
|-----------|-------|----------|
| Addisjon like termer | $ax + bx = (a+b)x$ | $3x + 5x = 8x$ |
| Multiplikasjon samme base | $x^m \times x^n = x^{m+n}$ | $x^2 \times x^3 = x^5$ |
| Divisjon samme base | $\frac{x^m}{x^n} = x^{m-n}$ | $\frac{x^5}{x^2} = x^3$ |
| Potens av potens | $(x^m)^n = x^{mn}$ | $(x^2)^3 = x^6$ |
| Potens av produkt | $(xy)^n = x^n y^n$ | $(2x)^3 = 8x^3$ |

## Øvingsoppgaver

**Addisjon og subtraksjon:**
- a) $4x + 7x$
- b) $8y - 3y$
- c) $2a + 5b + 3a$

**Multiplikasjon:**
- a) $x^3 \times x^4$
- b) $3a \times 4b$
- c) $(2x)^3$

**Divisjon:**
- a) $\frac{x^6}{x^2}$
- b) $\frac{12a^3}{4a}$

**Distributiv egenskap:**
- a) $3(x + 5)$
- b) $x(2x + 4)$
- c) $(x + 2)(x + 3)$

Bokstavregning danner grunnlaget for all videre algebraisk arbeid - når du mestrer disse reglene, er du klar for ligninger og mer avanserte emner!
